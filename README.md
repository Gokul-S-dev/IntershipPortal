# Internship Portal

A college portal application for managing internships with separate frontend and backend components.

## Project Structure

```
├── Backend/
│   └── demo.html          # Backend demo file
├── frontend/
│   ├── src/
│   │   ├── pages/         # React components for different pages
│   │   │   ├── AdminLogin.jsx
│   │   │   ├── Dashboard.jsx
│   │   │   ├── IndustryLogin.jsx
│   │   │   ├── Landing.jsx
│   │   │   ├── MentorLogin.jsx
│   │   │   ├── StudentLogin.jsx
│   │   │   └── StudentSignup.jsx
│   │   ├── App.jsx
│   │   ├── index.css
│   │   ├── index.jsx
│   │   └── main.jsx
│   ├── public/
│   ├── package.json
│   ├── vite.config.js
│   └── index.html
└── README.md
```

## Features

- Student Login and Signup
- Admin Login Portal
- Industry Login Portal
- Mentor Login Portal
- Dashboard for users
- Landing Page

## Technology Stack

- **Frontend**: React.js with Vite
- **Backend**: HTML (to be expanded)

## Getting Started

### Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
